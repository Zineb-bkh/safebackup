# SafeBackup
**SafeBackup** est un site web présentant une solution de **sauvegarde cloud sécurisée pour les entreprises.**
Le site permet de découvrir les fonctionnalités du service, les offres disponibles et de **contacter l'entreprise via un formulaire sécurisé.**

## Fonctionnalités
- Présentation des services de sauvegarde cloud
- Section fonctionnalités et avantages
- Plans et solutions avec différentes offres
- Formulaire de contact sécurisé en PHP
- Validation des données utilisateur
- Envoi d’email et sauvegarde des messages en JSON

## Technologies utilisées
- PHP
- HTML5
- CSS3
- JavaScript
- Font Awesome

## Utilisation
1. Placer le projet dans un serveur local (XAMPP / WAMP).
2. Ouvrir le fichier index.php dans le navigateur.
